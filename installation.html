The source code for this project can be found in this GitHub repository.

The easiest way to install beambending (requires pip installed) is:

python3 -m pip install --user git+https://github.com/alfredocarella/simplebendingpractice

Alternatively, you can try the web-based version online here, or clicking on the launch button.
